📋 使用说明摘要
1. 快速开始
   • 运行: python run_api.py
   • 配置: 首次运行会自动创建默认配置，需编辑config/config.json添加API密钥
   • 聊天: 在输入框输入内容，按Enter或点击发送按钮

2. 主要功能
   • 💬 聊天功能: 支持连续对话和上下文理解
   • 🔄 模型切换: 点击右上角切换AI模型
   • 🔊 语音播报: 勾选复选框开启，需配置百度TTS
   • 👤 角色管理: 支持添加多个角色，每个角色有独立聊天历史
   • 📊 响应式布局: 窗口大小变化时，角色卡片自动重新排列

3. 配置说明
   • API密钥: 配置openai.api_key
   • 模型设置: 在app.available_models中定义可切换模型
   • 外观设置: 配置background_image自定义背景

4. 注意事项
   • 确保已安装所有依赖: pip install tkinter pillow openai requests pygame
   • API密钥请妥善保管，不要分享给他人
   • 遇到问题可查看终端输出的错误信息进行排查
